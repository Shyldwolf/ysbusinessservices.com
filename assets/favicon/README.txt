Put this folder at: /assets/favicon/

Then add to <head> of every page:

<link rel="icon" href="/assets/favicon/favicon.ico" sizes="any">
<link rel="icon" type="image/png" href="/assets/favicon/favicon-32x32.png" sizes="32x32">
<link rel="icon" type="image/png" href="/assets/favicon/favicon-16x16.png" sizes="16x16">
<link rel="apple-touch-icon" href="/assets/favicon/apple-touch-icon.png" sizes="180x180">
<link rel="manifest" href="/assets/favicon/site.webmanifest">

Tip: Favicons cache aggressively. Hard refresh (Ctrl+F5) or test in Incognito.
