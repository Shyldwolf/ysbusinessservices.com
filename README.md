# YS Website (Static + Bilingual)

Static multi-page website built with **HTML/CSS/JS**, including a **language toggle** using `lang.js`.

## Pages
- Home, Services, About, FAQs, Testimonials, Privacy, Terms, Contact, Thanks

## Tech
- HTML / CSS / JavaScript
- Simple bilingual toggle logic
- Clean multi-page structure

## Run locally
Just open `index.html` in your browser.
